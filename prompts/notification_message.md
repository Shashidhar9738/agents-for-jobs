# Prompt: Notification Message

version: 1.0.0

Create a concise application event notification.

Inputs:
- Candidate id
- Company
- Role title
- Application status
- Match score
- Source portal

Rules:
- Keep it short and factual.
- Do not invent submission outcomes.
- Mention when an item needs manual review.

Output:
- One short message suitable for email or WhatsApp.
